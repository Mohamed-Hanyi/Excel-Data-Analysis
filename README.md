# Excel Data Analysis Project

This repository contains a comprehensive data analysis project focused on sales performance, customer insights, and operational metrics for an online gift delivery company (Ferns N Petals).

## Project Overview

The primary objective of this project was to transform raw data into actionable business insights. This included the complete ETL (Extract, Transform, Load) process to clean, model, and visualize data using Microsoft Excel.

## Key Features & Business Insights

- Revenue Analysis: Tracked total revenue across different months, occasions, and product categories.
- Customer Insights: Identified top-performing cities and customer purchasing behavior.
- Operational Performance: Calculated average delivery times to identify and address bottlenecks.
- Product Analysis: Analyzed top-selling products to optimize inventory and marketing efforts.
- Interactive Dashboard: Built a dynamic dashboard featuring slicers, charts, and key performance indicators (KPIs) for stakeholders to make informed decisions.

## Technical Workflow

1. Data Cleaning (ETL): 
   - Cleaned raw data by handling missing values and duplicates.
   - Standardized formatting for dates, phone numbers, and location data.
   - Removed irrelevant columns to streamline the dataset.
   
2. Data Transformation:
   - Used Power Query to merge tables (Customers, Orders, Products).
   - Created new calculated columns (Revenue, Month Name, Order Hour, Delivery Duration).

3. Data Modeling:
   - Designed a relational data model in Power Pivot, establishing relationships between fact and dimension tables.

4. Data Analysis:
   - Applied DAX functions to perform complex calculations and answer critical business questions.
   - Conducted correlation analysis to understand relationships between variables such as order quantity and delivery time.

## Tools & Technologies

- Microsoft Excel (Power Query, Power Pivot)
- Data Modeling & Relationships
- DAX (Data Analysis Expressions)
- Data Visualization (Pivot Tables & Charts)

## Conclusion

This project demonstrates the ability to handle the full data analysis lifecycle, from raw data preparation to providing strategic recommendations based on visual insights.
